## üêõ Common Issues & Solutions

### Issue: "No game available for testing"
**Solution:**
```bash
cd apps/api
pnpm db:seed
```

### Issue: Chat messages not syncing
**Check:**
1. All windows using same game ID?
2. API server running? (port 4301)
3. Check browser console for errors
4. Check network tab for SSE connection

### Issue: Scores not updating
**Check:**
1. User is registered/authenticated?
2. API server running?
3. Check POST request to `/api/games/{gameId}/score`
4. Check browser console for errors

### Issue: Can't test on mobile device
**Solution:**
1. Ensure mobile device on same WiFi network
2. Use your computer's local IP (not localhost)
3. Ensure firewall allows connections on port 4300
4. Try accessing http://[your-ip]:4300/test/complete-demo
